# Beta Kickoff Cadence (Day 30)

## Week 0 (Recruiting + Setup)
- Assign athlete IDs
- Confirm data connections (Intervals required; TP optional/coached)
- Set planner mode (coachMode on/off)

## Week 1 (Guided)
- **Day 1:** Setup + first recommendation + first feedback log
- **Day 3:** Midweek check-in (relevance, friction, confidence)
- **Day 7:** Retro (what helped, what confused, what to improve)

## Weekly operator rhythm
- **Monday:** Run metrics snapshot (`scripts/alpha_metrics_snapshot.py --write`)
- **Wednesday:** Review low relevance/high friction cases
- **Friday:** Ship 1-2 improvements and post changelog to cohort
